# Icones

Aquest add-in referencia les icones següents al `manifest.xml`. Afegeix aquí els PNG
corresponents abans de validar/publicar (durant el desenvolupament local, si falten,
Outlook mostrarà una icona genèrica):

- `icon-16.png` (16×16)
- `icon-32.png` (32×32)
- `icon-64.png` (64×64)
- `icon-80.png` (80×80)
- `icon-128.png` (128×128)

Poden ser el mateix logotip a diferents mides. Format PNG amb fons transparent recomanat.
